
# Moscato Vinoteca

Moscato es un proyecto de una vinoteca ubicada en Cordoba, Argentina.

Estamos desarrollando bajo el concepto de SPA

Libreria, frameworks y base de datos utilizados: 
- React
- Bootstrap 5
- React Icons
- Firebase

Los estilos se estan aplicando con SASS.

De momento la pagina cuenta con una navbar, catalogo, detalles de cada item, carrito de compras y navegabilidad por toda la pagina.

Aspectos a mejorar:
- Mejorar las cards del catalogo principal y vistas generales, alineando items y unificando una paleta de colores
- Hacer responsive toda la pagina
- Aprovechar mejor las opciones que nos da SASS creando variables, mixins y maps. 

Funciones a crear:
